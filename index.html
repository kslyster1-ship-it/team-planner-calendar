Enable Multi-User Collaboration
This app needs a free Firebase Realtime Database to sync between users in real time. Without it, data is stored locally in your browser only.
1
Go to console.firebase.google.com and create a new project (free)
2
In your project: Build → Realtime Database → Create database (choose "test mode")
3
Go to Project Settings → General → Your apps → Add app (Web)
4
Copy the firebaseConfig object and click "Configure Firebase" below to paste it in
5
Deploy this HTML file to Netlify Drop (drag & drop — takes 30 seconds) and share the URL
